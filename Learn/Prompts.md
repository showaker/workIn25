### 🔧 1. **刷算法题时卡住了（看不懂题 / 不会写 / 超时）**
```prompt
你是一位经验丰富的算法教练，擅长用清晰的步骤引导初学者。请按以下结构帮助我：
1. **题目复述**（用我自己的话简述问题，确认理解正确）
2. **核心难点分析**（指出关键约束、易错点、常见误区）
3. **暴力解法 → 优化思路**（分步说明如何想到更优解，如双指针、哈希、动态规划等）
4. **代码实现**（Python，带详细注释，时间/空间复杂度标注）
5. **测试用例设计**（含边界情况：空输入、重复元素、极端值等）

我现在的问题是：[在此粘贴题目 / 描述卡住的具体点]
```

> ✅ 效果：避免模型直接甩代码，强调“思考过程”，适合学习内化。

### 💻 2. **代码报错 / 行为异常**
```prompt
我正在用 [语言，如Python 3.11] 写一段代码，预期行为是 [简述目标]，但实际运行时：
- 报错信息：[粘贴完整报错]
- 或：输出为 [实际输出]，但期望是 [期望输出]

请：
1. 定位问题根源（指出具体哪行、为什么错）
2. 解释相关知识点（如：可变默认参数陷阱、闭包变量绑定、GIL影响等）
3. 给出修正后的代码（保留原风格，最小改动）
4. 提供1–2个同类易错点的预防建议

我的代码如下：
```[语言]
[粘贴代码]
```
```

> ✅ 效果：引导模型做“debug教练”，不只是修bug，更要防bug。

### 📚 3. **想深入理解某个算法/数据结构**
```prompt
请以“费曼教学法”向我讲解 [算法/数据结构名称，如：红黑树 / KMP / 快速选择]：
1. **一句话本质**：用生活类比说明它解决什么问题
2. **核心思想图解**：用文字描述关键步骤（如旋转、回溯指针、分区）
3. **为什么高效**：对比朴素方法，分析时间/空间优势
4. **典型应用场景**：举2个真实工程例子（如数据库索引、文本搜索）
5. **常见面试变形题**：列出1–2道变体及解题线索

最后，用 [Python/Java] 写一个极简可运行的核心实现（不求完整，但关键逻辑清晰）。
```

> ✅ 效果：避免学术化堆砌，强调直觉理解 + 实用性。
